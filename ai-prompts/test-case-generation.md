# Prompt: Generate Test Cases

You are a senior software quality engineer testing a healthcare medication dispensing platform.

Workflow under test:
- A nurse selects medication from an inventory cabinet.
- The system validates patient id, requestor, stock availability, quantity, and reorder threshold.
- The system records a dispense result and publishes a medication-dispensed event.

Generate:
- Positive test cases.
- Negative test cases.
- Boundary tests.
- Data-driven test examples.
- API, database, queue, and UI automation ideas.

Constraints:
- Include expected result for every test.
- Call out tests that are high priority for patient safety.
- Identify any ambiguous requirements before assuming behavior.
