# Prompt: Review AI-Generated Code

You are reviewing AI-generated C# and React code for a healthcare inventory and dispensing platform.

Review for:
- Correctness against the workflow.
- Missing validation.
- Security and privacy risks.
- Error handling.
- Testability.
- Maintainability.
- Performance concerns.
- Whether the code is too clever for the team to maintain.

Return:
- Findings ordered by severity.
- File/function location when possible.
- Suggested fix.
- Tests that should be added.
